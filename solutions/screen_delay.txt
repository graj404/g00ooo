Commands:

vel 20      - Set 20 m/s velocity

quit        - Exit

sudo python3 test_hardware_interrupts.py

Success rate
Average interval

3. README_HARDWARE_TEST.md - Complete Guide
Hardware Setup (Super Simple!):
Raspberry Pi 5
┌─────────────────────┐
│                     │
│  GPIO 27 (Output) ──┼──┐  ← Generates pulses
│                     │  │
│                     │  │  One wire!
│                     │  │
│  GPIO 17 (Input)  ──┼──┘  ← Receives pulses
│                     │
└─────────────────────┘
Just connect GPIO 27 → GPIO 17 with a wire!

How to Test Your System:
Terminal 1 - Generate Pulses:
cd Map_Test
sudo python3 pulse_generator.py
[Current: 0 RPM] > vel 20
Terminal 2 - Run Dead Reckoning:
cd Lap_map_main
sudo python3 main.py
Watch it track position in real-time!

Why This Works:
The pulse goes through the EXACT same hardware path as the real hall sensor:

GPIO 27 → GPIO 17 → BCM2712 Interrupt Controller → 
Linux Kernel → Python Callback (0.01ms total!)
