# Test Cases for Dead Reckoning Calculations

## Test Case 1: Straight Line East
Input:
  velocity = 10 m/s
  heading = 0° (0 rad)
  steering = 0°
  dt = 0.01 s

Expected Output:
  dx = 0.1 m
  dy = 0.0 m
  new_position = (0.1, 0.0)

## Test Case 2: Straight Line North
Input:
  velocity = 10 m/s
  heading = 90° (1.5708 rad)
  steering = 0°
  dt = 0.01 s

Expected Output:
  dx ≈ 0.0 m
  dy ≈ 0.1 m
  new_position ≈ (0.0, 0.1)

## Test Case 3: Straight Line Northeast
Input:
  velocity = 10 m/s
  heading = 45° (0.7854 rad)
  steering = 0°
  dt = 0.01 s

Expected Output:
  dx ≈ 0.0707 m
  dy ≈ 0.0707 m
  new_position ≈ (0.0707, 0.0707)

## Test Case 4: Right Turn
Input:
  velocity = 5 m/s
  heading = 0° (0 rad)
  steering = 15° (0.2618 rad)
  dt = 0.01 s
  wheelbase = 1.0 m

Expected Output:
  heading_rate ≈ 1.339 rad/s
  new_heading ≈ 0.0134 rad (0.77°)
  dx ≈ 0.0499 m
  dy ≈ 0.0007 m

## Test Case 5: Left Turn
Input:
  velocity = 5 m/s
  heading = 0° (0 rad)
  steering = -15° (-0.2618 rad)
  dt = 0.01 s
  wheelbase = 1.0 m

Expected Output:
  heading_rate ≈ -1.339 rad/s
  new_heading ≈ -0.0134 rad (-0.77°)
  dx ≈ 0.0499 m
  dy ≈ -0.0007 m

## Test Case 6: High Speed
Input:
  velocity = 50 m/s
  heading = 0° (0 rad)
  steering = 0°
  dt = 0.01 s

Expected Output:
  dx = 0.5 m
  dy = 0.0 m
  new_position = (0.5, 0.0)

## Test Case 7: RPM to Velocity
Input:
  rpm = 1800
  wheel_diameter = 0.2 m
  wheel_circumference = 0.6283 m

Expected Output:
  velocity = (1800 / 60) × 0.6283
  velocity = 30 × 0.6283
  velocity ≈ 18.85 m/s

## Test Case 8: Multiple Updates (Circle)
Input:
  velocity = 5 m/s
  heading = 0° initially
  steering = 30° (0.5236 rad)
  dt = 0.01 s
  updates = 100

Expected Output:
  Should trace a circular arc
  Final heading ≈ 90° (quarter circle)
  Approximate radius ≈ 2 m

## Test Case 9: Zero Velocity
Input:
  velocity = 0 m/s
  heading = 45°
  steering = 15°
  dt = 0.01 s

Expected Output:
  dx = 0.0 m
  dy = 0.0 m
  new_position = (0.0, 0.0)
  heading unchanged

## Test Case 10: Angle Normalization
Input:
  heading = 350° (6.1087 rad)
  add 30° (0.5236 rad)

Expected Output:
  raw_heading = 380° (6.6323 rad)
  normalized_heading = 20° (0.3491 rad)
  (380° - 360° = 20°)
